## Build Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for staging with minification
npm run build:staging

# check eslint script
npm run lint:script

# check eslint style
npm run lint:style

# fix eslint script
npm run lint:fix
```
